# ML Big Data Analytics Project

## Overview
This project provides a comprehensive scaffold for implementing a machine learning and data analytics solution using PySpark and scikit-learn tailored for big dataset workflows.

## Architecture
The architecture consists of multiple components that manage data ingestion, cleaning, feature engineering, modeling, evaluation, deployment, and monitoring.

## Dataset Setup
The dataset used for this project is the NYC Taxi dataset available on Kaggle.

## Workflow
1. **Problem**
2. **Data**
3. **Cleaning**
4. **Feature Engineering**
5. **Exploratory Data Analysis (EDA)**
6. **Model**
7. **Evaluation**
8. **Deployment**
9. **Monitoring**

## Step-by-step Commands
- Install dependencies: `pip install -r requirements.txt`
- Run the pipeline: `python scripts/run_pipeline.py`
- Launch the Streamlit dashboard: `streamlit run dashboard/app.py`
